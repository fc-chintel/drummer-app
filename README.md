# 🥁 ドラムコーチ — リズムキープ・トレーナー

ドラム初心者のためのリズム練習 Web アプリ。メトロノームに合わせて叩くと、
タイミングのズレ(走り気味/もたり気味)をリアルタイムで表示・採点します。

## 機能

- **正確なメトロノーム** — Web Audio のルックアヘッド方式でズレないクリック
- **タイミング判定** — パーフェクト(±35ms)/グッド(±70ms)/おしい(±120ms)、コンボとスコア
- **ズレメーター** — 1発ごとに「早い/遅い」を何ミリ秒単位で表示
- **ギャップクリック** — クリックが数小節ごとに消えるモード。無音の間もキープできるかが試される、リズムキープ練習の定番
- **セッションまとめ** — 平均のズレ・ばらつき(標準偏差)と一言アドバイス。記録はブラウザに保存
- **入力は3方式**
  - マイク: どの端末でも動く(iPhone/iPad 含む)
  - MIDI: 電子ドラム(Donner DED-300X など)を USB 接続。PC / Android の Chrome で使える。精度が高いのでおすすめ
  - タップ: 画面タップ or スペースキー。機材なしでどこでもリズム練習&動作確認用

## 動かし方

```
npm install
npm run dev
```

表示された URL を開く。PC なら `https://localhost:5173`。

### スマホ/タブレットから使う

1. PC と同じ Wi-Fi につなぐ
2. `npm run dev` の出力に出る `https://192.168.x.x:5173` をスマホで開く
3. 証明書の警告が出たら「詳細設定」→「アクセスする」で進む(開発用の自己署名証明書のため)
4. マイクの使用を許可する

### 電子ドラム(MIDI)で使う

1. DED-300X の USB 端子と PC(または Android)をケーブルで接続
2. 入力方式を「MIDI」にしてスタート
3. 画面に「接続中: …」と出れば OK

## 使い方のコツ

- 最初は BPM 80〜90 の 4/4 で、8ビートかスネアだけで試す
- 「遅れ補正」: ジャストに叩いているのに遅い判定が出るときは値を増やす
- マイクで誤検出が多いときは「マイク感度」の数字を大きくする(鈍感になる)
- 慣れてきたらギャップクリック「2小節鳴らす/2小節消す」に挑戦

## インターネットに公開する(どこからでも・誰でも使えるように)

このアプリはサーバー不要(全部ブラウザ内で完結、記録もブラウザ保存)なので、
無料の静的ホスティングに置くだけで公開できます。

1. GitHub にこのリポジトリをプッシュする
2. [Netlify](https://www.netlify.com/) か [Vercel](https://vercel.com/)(どちらも無料枠で十分)にGitHubアカウントでログインし、リポジトリを選ぶ
3. ビルド設定はそのまま(build コマンド `npm run build`、公開ディレクトリ `dist`)
4. `https://○○○.netlify.app` のようなURLが発行される — これを共有するだけ

公開URLは自動的にHTTPSなので、マイクもMIDIもそのまま動きます。

### PWA(ホーム画面に追加)

このアプリはPWA対応済みです。公開URLをスマホで開いて:

- **Android (Chrome)**: メニュー →「ホーム画面に追加」または「アプリをインストール」
- **iPhone/iPad (Safari)**: 共有ボタン →「ホーム画面に追加」

するとアイコンから全画面のアプリとして起動でき、一度開けばオフラインでも動きます
(練習記録は端末ごとのブラウザに保存されます)。

## これからのアイデア(第2弾以降)

- ドラム譜(TAB譜)エディタ: 好きな曲のドラムパートを打ち込んで再生・練習
- 譜面を見ながら演奏 → 譜面どおり叩けたか判定(リズムゲーム風)
- 自動キャリブレーション(遅れ補正を自動測定)
- 練習記録のグラフ表示
